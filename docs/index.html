macOS Menu Bar Utility

VBRecorder

Capture the selected word from any app and append it to a local CSV file. Built for a lightweight, keyboard-first workflow.

Download Latest DMG View Release View Source

Install

Download VBRecorder.dmg, drag the app into Applications, then launch it from Finder or Spotlight.

Permission

On first use, enable Accessibility for the app in System Settings so it can read selected text from other apps.

Storage

Words are stored in a CSV file with timestamps and rank fields for first add order and recent access order.

Builds

The main-latest prerelease is refreshed automatically by GitHub Actions on every push to main.
